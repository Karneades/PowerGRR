# Stop-GRRHunt

## SYNOPSIS
Stop a hunt.

## SYNTAX

```
Stop-GRRHunt [[-HuntId] <String>] [[-Credential] <PSCredential>] [-ShowJSON] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Stop a hunt.

## EXAMPLES

### Example 1
```
PS C:\> Stop-GRRHunt -Credential $cred -HuntId 7D5E2318
```

Stop the specific hunt.
